How it works

  1. Pick the architecture (s390 or x86).
  2. Click Open Listing to load addresses and statements.
  3. Check the module name, then click Open Trace to build the mapping.
  4. Save Map stores the full mapping; Save Source Trace stores only the source lines.

Works in Google Chrome and Microsoft Edge.
